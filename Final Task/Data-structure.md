# Структура данных

Структура таблицы заболевших:
* фамилия строка,
* имя строка,
* отчество строка,
* дата выявления дата,
* дата выписки дата,
* состояние(выздоровел, болеет, погиб) строка

Структура xml
```xml
<statistic>
    <start-date>дата начала</start-date>
    <end-date>дата конца</end-date>
    <ratio>соотношение</ratio>
</statistic>
```
