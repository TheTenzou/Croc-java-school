# Структура данных

Структура таблицы заболевших:
* first_name string,
* midle_name string,
* last_name string,
* start_date date,
* end_date date,
* condition(recover, sick, died) string

Структура xml
```xml
<statistic>
    <start-date>дата начала</start-date>
    <end-date>дата конца</end-date>
    <ratio>соотношение</ratio>
</statistic>
```
